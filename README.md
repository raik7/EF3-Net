# EF3-Net

2K-Fold-Net and Feature Enhanced 4-Fold-Net for Medical Image Segmentation.    

Run train.py to train the model.  
Run predict.py to generate the predicted segmentation.  

## Requirement
tensorflow 2.0.1  
numpy 1.20.1  
cv2 4.5.1  
scikit-image 0.18.1  
scikit-learn 0.24.1  
tqdm 4.58.0  
libtiff 0.4.2
